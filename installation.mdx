## Overview

Erdo provides multiple ways to get started, depending on your use case and preferences. This guide covers installation of the CLI, SDK, and platform access.

## Erdo CLI

The CLI is the fastest way to build and deploy Erdo agents.

### Homebrew (Recommended)

<CodeGroup>

```bash macOS/Linux
brew install erdoai/tap/erdo
```

```bash Update
brew upgrade erdo
```

</CodeGroup>

### Manual Installation

Download the latest release for your platform:

<CodeGroup>

```bash macOS (Intel)
curl -L https://github.com/erdoai/homebrew-tap/releases/latest/download/erdo-cli_Darwin_x86_64.zip -o erdo-cli.zip
unzip erdo-cli.zip
chmod +x erdo
sudo mv erdo /usr/local/bin/
rm erdo-cli.zip
```

```bash macOS (Apple Silicon)
curl -L https://github.com/erdoai/homebrew-tap/releases/latest/download/erdo-cli_Darwin_arm64.zip -o erdo-cli.zip
unzip erdo-cli.zip
chmod +x erdo
sudo mv erdo /usr/local/bin/
rm erdo-cli.zip
```

```bash Linux (x86_64)
curl -L https://github.com/erdoai/homebrew-tap/releases/latest/download/erdo-cli_Linux_x86_64.zip -o erdo-cli.zip
unzip erdo-cli.zip
chmod +x erdo
sudo mv erdo /usr/local/bin/
rm erdo-cli.zip
```

```bash Windows
# Download erdo-cli_Windows_x86_64.zip from GitHub Releases
curl -L https://github.com/erdoai/homebrew-tap/releases/latest/download/erdo-cli_Windows_x86_64.zip -o erdo-cli.zip
# Extract erdo.exe and add to PATH
```

</CodeGroup>

### Verify Installation

```bash
erdo --version
```

## Python SDK

Install the Erdo Python SDK for pure Python workflows - sync, test, and invoke agents all from Python without needing the CLI.

<CodeGroup>

```bash pip
pip install erdo
```

```bash Poetry
poetry add erdo
```

```bash uv
uv add erdo
```

</CodeGroup>

### Verify SDK Installation

```python
import erdo
from erdo.sync import Sync
from erdo.invoke import Invoke

print(f"Erdo SDK version: {erdo.__version__}")
print("SDK modules available: sync, invoke, test")
```

### SDK Configuration

The SDK uses the same configuration as the CLI, reading from `~/.erdo/config.yaml` or environment variables:

```python
from erdo.config import Config

# Automatically loads from ~/.erdo/config.yaml or env vars
config = Config()
print(f"API URL: {config.api_url}")
print(f"Authenticated: {config.is_authenticated()}")
```

## Platform Access

### Create Account

1. Go to [erdo.ai](https://erdo.ai)
2. Sign up for a free account
3. Verify your email address

### Authentication

<Tabs>
  <Tab title="CLI">
    Connect your local tools to the platform:

    ```bash
    erdo login
    ```

    This will:
    - Open your browser for authentication
    - Store your credentials securely
    - Set up API access for both CLI and SDK

    ### Verify Authentication
    ```bash
    erdo whoami
    ```
  </Tab>
  <Tab title="SDK">
    The SDK automatically uses CLI authentication. If you haven't logged in yet:

    ```bash
    erdo login
    ```

    Or set environment variables directly:
    ```bash
    export ERDO_API_URL="https://api.erdo.ai"
    export ERDO_AUTH_TOKEN="your-token"
    ```

    ### Verify in Python
    ```python
    from erdo.config import Config
    
    config = Config()
    if config.is_authenticated():
        print("Ready to sync and invoke agents!")
        print(f"Organization: {config.organization}")
    else:
        print("Please run 'erdo login' first")
    ```
  </Tab>
</Tabs>

## Configuration

### Environment Variables

Set up common environment variables:

```bash
# ~/.bashrc or ~/.zshrc
export ERDO_API_URL="https://api.erdo.ai"
export ERDO_LOG_LEVEL="info"
export ERDO_CACHE_DIR="~/.erdo/cache"
```

### Configuration File

Create a global configuration file at `~/.erdo/config.yaml`:

```yaml
# Erdo Configuration
api:
  url: "https://api.erdo.ai"
  timeout: 30

logging:
  level: "info"
  format: "structured"

cache:
  enabled: true
  ttl: 3600

development:
  auto_save: true
  watch_mode: true
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="CLI installation fails">
    **Permission denied errors:**
    ```bash
    sudo chmod +x erdo
    sudo chown $(whoami) erdo
    ```

    **PATH issues:**
    ```bash
    echo 'export PATH="/usr/local/bin:$PATH"' >> ~/.bashrc
    source ~/.bashrc
    ```

  </Accordion>

  <Accordion title="SDK import errors">
    **Virtual environment issues:**
    ```bash
    python -m venv erdo-env
    source erdo-env/bin/activate  # On Windows: erdo-env\Scripts\activate
    pip install erdo
    ```

    **Dependency conflicts:**
    ```bash
    pip install --upgrade erdo
    pip install --force-reinstall erdo
    ```

  </Accordion>

  <Accordion title="Authentication problems">
    **Login failures:**
    ```bash
    erdo logout
    erdo login --force
    ```

    **Token issues:**
    ```bash
    rm ~/.erdo/auth.json
    erdo login
    ```

  </Accordion>

  <Accordion title="Performance issues">
    **Slow operations:**
    - Clear cache: `erdo cache clear`
    - Update CLI: `brew upgrade erdo`
    - Check internet connection
    - Verify system resources
  </Accordion>
</AccordionGroup>
